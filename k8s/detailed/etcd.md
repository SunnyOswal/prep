# ETCD : Distributed reliable key-value store that is simple,secure & fast.

+ key-value store
    + Stores info in the form of documents or pages.
    + unstructured.
    + JSON/YAML
+ installation
    + Download binary
    + extract
    + run etcd service
+ Operate
    + PORT 2379
    + etcdctl client/CLI used.
    + ectdl get/set common cmds.
